# 🧠 Intelligent CPU Scheduler Simulator  
A modern web-based simulator for visualizing CPU scheduling algorithms in real time.  
Built with **React + TypeScript + Vite**, styled using **Tailwind CSS** and **shadcn/ui**.

---

## 🔗 Live Demo  
👉 **View Live Project:** https://os-project-rust.vercel.app/  


---

## 📚 Project Overview  
This project simulates major CPU scheduling algorithms with dynamic visualizations, process handling, and performance metrics.  
It helps users understand real OS scheduling behavior through a clean, interactive UI.

---

## 🧩 Features  
- 🎛️ Input custom processes (arrival, burst, priority)  
- ⚙️ Supports **major scheduling algorithms**:  
  - FCFS (First-Come First-Serve)  
  - SJF (Shortest Job First – Preemptive/Non-preemptive depending on design)  
  - Priority Scheduling  
  - Round Robin  
- 📊 Real-time Gantt chart visualization  
- 📈 Shows Waiting Time, Turnaround Time, Completion Time  
- 🌙 Beautiful UI using shadcn components  
- ⚡ Fast performance using Vite
